# React + Recoil + React Query + React Hooks + Typescript + Axios + Vite - Starter

This Starter utilizes React, Recoil, React Query, React Hooks, Typescript, Axios And Vite.

## [中文文档](./README_ZH.md)




## Installation

Clone this repo and install all dependencies

```
 yarn install
```

## Development

``` 
yarn dev 
```

## Build

```
yarn build
```

## Publish

```
yarn dist
```
